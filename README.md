# sig-nal

&lt;sig-nal&gt; is a zero-dependencies Web Component for Server-Side Rendering with Signals (SSRS) in [**1543**](https://raw.githubusercontent.com/cloudspeech/sig-nal/main/dist/index.js) bytes (minified + Brotli-11-compressed).

## Counter Example

No build needed, 27 lines of static HTML, 3 kB. Compares favourably to [Fresh framework by Deno](https://fresh.deno.dev/) and others! [Try it out!](https://cloudspeech.github.io/sig-nal/counter.html)
